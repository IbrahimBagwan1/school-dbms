# **App Name**: ScholarSight

## Core Features:

- Student Information Management: Manage student records, including personal details, contact information, enrollment history, and academic performance.
- Teacher Information Management: Manage teacher profiles, including personal details, contact information, subject expertise, and employment history.
- Course and Class Scheduling: Create and manage courses, class schedules, and student-teacher assignments.
- Gradebook and Performance Tracking: Record and track student grades, attendance, and overall academic performance.
- Predictive Failure Analysis: Employ a tool using AI algorithms to analyze student performance data and predict students at risk of failing upcoming tests, based on historical grades, attendance, and other relevant factors.
- Report Generation: Generate reports on student performance, attendance, and other key metrics.
- User Authentication and Authorization: Secure access to the system with role-based permissions for students, teachers, and administrators.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust, intelligence, and stability.
- Background color: Very light gray (#F5F5F5) to provide a clean, professional backdrop.
- Accent color: Vibrant orange (#FF9800) for calls to action and important notifications.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and 'Inter' (sans-serif) for body text.
- Use clean, professional icons from a consistent set (e.g., Material Design Icons).
- Implement a clear and intuitive layout with easy navigation and a responsive design that adapts to different screen sizes.
- Use subtle animations to provide feedback on user interactions and guide users through the system.
